API 安全性是现代 Web 开发中不可忽视的环节。以下是常见的安全威胁和防御策略。

## 认证与授权

使用 JWT 或 OAuth 2.0 进行身份认证，确保每个请求都经过验证。

## 常见攻击

- SQL 注入：使用参数化查询
- XSS：对用户输入进行转义
- CSRF：使用 Token 机制
- 速率限制：防止暴力破解

## 最佳实践

始终使用 HTTPS，实施最小权限原则，定期审计 API 日志。
