Rust 的所有权系统是其最独特也最重要的特性。它让 Rust 能够在没有垃圾回收器的情况下保证内存安全。

## 所有权规则

Rust 的所有权遵循三条简单规则：

1. 每个值在 Rust 中都有一个所有者
2. 同一时间只能有一个所有者
3. 当所有者离开作用域时，值将被丢弃

## 借用与引用

除了移动所有权，Rust 还允许通过引用借用的方式访问数据：

```rust
fn calculate_length(s: &String) -> usize {
    s.len()
}
```

引用允许你访问值而不获取其所有权。这在函数参数中非常常见。
